# h4ckb0x
My simple, docker based, hacking environment FTW.

This Dockerfile defines a comprehensive environment intended as the base for an offensive security Docker image.
It bundles reconnaissance, scanning, exploitation, and other offensive tools to support penetration testing workflows. 
It was created as the foundation for preparing for the HTB CPTS certification and helped me pass the exam.


## Build

Just use the `make` to build the container image.

```bash
$ make
```

## Run

```bash
# Define a project dir
$ export HACKBOX_PROJECT_DIR="/myProjectDir"

# Optional: Define a list of env variables you would like to take set into the h4ckb0x (for example $IP as your primary CTF target)
$ echo "IP=127.0.0.1" >> $HOME/.h4ckb0x.env

# Start hacking
$ h4ckb0x
```
## Content - Tools included in the h4ckb0x Dockerfile (categorized)

### Reconnaissance & subdomain / OSINT
- `amass` - DNS and asset discovery, subdomain enumeration.
- `Sublist3r` - subdomain discovery.
- `subbrute` - subdomain brute-forcer.
- `dnsenum` - DNS enumeration tool.
- `whois` - domain registration lookups.
- `whatweb` - website fingerprinting.
- `EyeWitness` - capture screenshots and identify interesting web pages and services.
- `FinalRecon` - automated reconnaissance aggregation framework.
- `n0kovo_subdomains` (wordlist repo) - domain/subdomain wordlists.
- `statistically-likely-usernames` (wordlist repo) - probable username lists.
- `SecLists` - curated set of wordlists for discovery and fuzzing.
- `smtp-user-enum.pl` - SMTP username enumeration script.
- `nmap` (installed/built) - network and service discovery.

### Network & port scanning / enumeration
- `nmap` - port and service scanning and version detection.
- `nikto` - web server vulnerability scanner.
- `smbclient` - SMB client for enumeration and interaction.
- `onesixtyone` - SNMP enumeration.
- `snmp` utilities - SNMP enumeration and checks.
- `mysql-client` / `sqlite3` / `sqsh` - database clients used for enumeration.
- `ncat` / `netcat` / `socat` - TCP and UDP connection, tunneling and relaying utilities.
- `rdp-sec-check` - RDP specific checks.

### Web application testing & CMS scanning
- `sqlmap` - automated SQL injection discovery and exploitation.
- `SSTImap` - automated SSTI injection discovery and exploitation.
- `JoomlaScan` - Joomla CMS scanner.
- `smuggler` - HTTP request smuggling testing tool.
- `whatweb` - web fingerprinting.
- `EyeWitness` - visual triage for web services.

### Fuzzing, wordlists & content discovery helpers
- `SecLists` - large collection of wordlists (directories, usernames, passwords).
- `rockyou` (wordlists) - common password list.
- `fff` (tomnomnom/fff) - fast file and wordlist filtering utilities.
- `gf` (tomnomnom/gf) - grep patterns and recipes for fuzzing output.
- `anew` - append only new lines to files (wordlist workflow).
- `cupp` - common user password profiler.
- `cewl` - custom wordlist generator from websites.

### Password cracking & brute-forcing
- `hydra` - network login brute-forcer.
- `medusa` - parallel login brute-forcer.
- `hashcat` - GPU/CPU password cracking tool.
- `john` (John the Ripper) - password cracking suite.
- `PCredz` - credential extraction and helper utilities.

### Windows / Active Directory enumeration & exploitation
- `impacket` - Python library and tools for SMB, NTLM and AD interactions.
- `Responder` - LLMNR, NetBIOS and mDNS poisoning and credential capture.
- `windapsearch` - LDAP enumeration helper.
- `nishang` - PowerShell scripts for post-exploitation.
- `PowerSploit` - PowerShell offensive toolkit.
- `pyGPOAbuse` - GPO abuse enumeration and exploitation.
- `PKINITtools` - tools for certificate-based Kerberos attacks.
- `targetedKerberoast` - Kerberoasting helpers.
- `PetitPotam` - Active Directory NTLM relay and abuse PoCs.
- `rdp-sec-check` - RDP specific checks and enumeration.

### Post-exploitation, privilege escalation & payload helpers
- `Invoke-DOSfuscation` - PowerShell obfuscation scripts.
- `Bashfuscator` - obfuscate Bash payloads and scripts.
- `PCredz` - credential harvesting helpers.
- `FinalRecon` - aggregates reconnaissance output for follow-up.

### Tunneling, pivoting & covert channels
- `chisel` - fast TCP/UDP tunnel over HTTP.
- `dnscat2` - DNS-based command channel and covert C2.
- `ptunnel-ng` - TCP over ICMP or other covert channels.
- `rpivot` - remote pivoting helper scripts.

### Binary analysis & exploit development

- various exploit PoCs (for example `CVE-2021-1675` repo) - proof of concept exploit code included.

### Forensics, auditing & system checks
